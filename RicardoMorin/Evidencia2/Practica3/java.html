¿Qué es Java?

Java es un lenguaje que se usa en todo el mundo, desde aplicaciones móviles hasta grandes sistemas. Es como el café: ¡todo el mundo lo ama!

Tipos de Datos

  • Enteros
  • Flotantes
  • Cadenas
  • Arreglos
  • Objetos

Ventajas

  • Portabilidad: escribe una vez, corre en cualquier lugar.
  • Gran comunidad y muchas bibliotecas disponibles.

Desventajas

  • Puede ser más lento que otros lenguajes.
  • Consumo de memoria puede ser alto.
