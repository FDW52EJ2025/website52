¿Qué es Python?

Python es un lenguaje super popular y fácil de usar. Es como el amigo que todos quieren tener: versátil y siempre listo para ayudar.

Tipos de Datos

  • Enteros (números enteros, ¡obvio!)
  • Flotantes (números con decimales, como 3.14)
  • Cadenas (texto, como "Hola Mundo")
  • Listas (colecciones de cosas, como tus juegos favoritos)
  • Diccionarios (pares clave-valor, como un mini Google)

Ventajas

  • Súper fácil de aprender, ¡ideal para principiantes!
  • Gran comunidad, siempre hay alguien para ayudar.
  • Se usa en todo, desde web hasta ciencia de datos.

Desventajas

  • Puede ser más lento que otros lenguajes, pero ¡no te preocupes!
  • Consume más memoria, así que cuida tu computadora.
