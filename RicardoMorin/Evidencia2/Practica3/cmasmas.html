¿Qué es C++?

C++ es un lenguaje que se basa en C, pero con un toque extra. Es como el hermano mayor que sabe más y puede hacer más cosas.

Tipos de Datos

  • Enteros
  • Flotantes
  • Cadenas
  • Vectores
  • Estructuras

Ventajas

  • Rápido y eficiente, ideal para juegos y aplicaciones grandes.
  • Control total sobre el hardware.

Desventajas

  • Más complicado de aprender que otros lenguajes.
  • Puede ser propenso a errores si no tienes cuidado.

Ejemplo "Hola Mundo"

#include <iostream>

int main() {
    std::cout << "Hola Mundo" << std::endl;
    return 0;
}
