JavaScript

JavaScript (abreviado conmunmente JS) es un lenguaje de programación interpretado, Se define como orientado a objetos, basado en prototipos, imperativo, débilmente tipado y dinámico.

Se utiliza principalmente del lado del cliente, implementado como parte de un navegador web permitiendo mejoras en la interfaz de usuario y páginas web dinámicas[3]​ y JavaScript del lado del servidor (Server-side JavaScript o SSJS). Su uso en aplicaciones externas a la web, por ejemplo en documentos PDF, aplicaciones de escritorio (mayoritariamente widgets) es también significativo.

Caracteristicas principales:

  • Interpretado, dinámico y basado en prototipos.
  • Tipado débil y flexible.
  • Soporta OOP, funciones de orden superior, asincronía con async/await y promesas.
  • Muy integrado con HTML y el navegador.

Usos comunes

  • Desarrollo web frontend (React, Vue, Angular).
  • Backend con Node.js.
  • Aplicaciones móviles híbridas y desarrollo de APIs.

Tipos de datos:

1.Primitivos

  • Number: enteros y flotantes (42, 3.14)
  • BigInt: enteros grandes (123n)
  • String: cadenas ("Hola")
  • Boolean: true, false
  • undefined: valor no asignado
  • null: valor intencionalmente nulo
  • Symbol: identificador único

2.Estructurados

  • Object: colección de pares clave-valor
  • Array: listas ordenadas
  • Function: funciones tratadas como datos

Ventajas

  • El único lenguaje nativo en todos los navegadores.
  • Gran comunidad y ecosistema.
  • Multiplataforma con Node.js.
  • Ideal para desarrollo rápido.

Desventajas

  • Tipado débil puede causar errores difíciles.
  • Inconsistencias entre navegadores (menos hoy).
  • Algunas APIs tienen comportamiento inesperado.
