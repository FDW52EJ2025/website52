Java

Java es un lenguaje de programciony una plataforma informatica que fue comercializada por primera vez en 1995 por Sun Microsystems.

Java fue desarrollado originalmente por James Gosling, de Sun Microsystems (constituida en 1983 y posteriormente adquirida el 27 de enero de 2010 por la compañía Oracle), y publicado en 1995 como un componente fundamental de la plataforma Java de Sun Microsystems. Su sintaxis deriva en gran medida de C y C++, pero tiene menos utilidades de bajo nivel que cualquiera de ellos. Las aplicaciones de Java son compiladas a bytecode (clase Java), que puede ejecutarse en cualquier máquina virtual Java (JVM) sin importar la arquitectura de la computadora subyacente.

Caracteristicas principales:

  • Lenguaje compilado a bytecode, que se ejecuta en la JVM.
  • 100% orientado a objetos.
  • Tipado estático y fuerte.
  • Recolección automática de basura.
  • Amplia API estándar (colecciones, IO, GUI, etc.).

Tipos de datos:

1.Primitivos

  • byte (8 bits)
  • short (16 bits)
  • int (32 bits)
  • long (64 bits)
  • float (32 bits decimales)
  • double (64 bits decimales)
  • char: caracteres Unicode ('A')
  • boolean: true o false

2.Referenciados

  • String: texto ("Hola")
  • Array, Object, List, Map, etc.
  • Cualquier clase definida por el usuario

Ventajas

  • Portabilidad total ("Write once, run anywhere").
  • Robusto y seguro.
  • Gran ecosistema (Spring, Hibernate, Maven…).

Desventajas

  • Código extenso y verboso.
  • Consumo de memoria elevado.
  • Lentitud en la JVM frente a código nativo.
