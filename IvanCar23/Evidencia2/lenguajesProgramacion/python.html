Python

Python es un lenguaje de alto nivel de programación interpretado cuya filosofía hace hincapié en la legibilidad de su código. Se trata de un lenguaje de programación multiparadigma, ya que soporta parcialmente la orientación a objetos, programación imperativa y, en menor medida, programación funcional. Es un lenguaje interpretado, dinámico y multiplataforma.

Creado por Guido van Rossum y lanzado en 1991, Python fue diseñado para ser fácil de leer y escribir. Su nombre proviene del grupo cómico británico Monty Python, no de la serpiente. Es muy usado en ciencia de datos, inteligencia artificial, desarrollo web, automatización y scripting.

Caracteristicas principales:

  • Interpretado, de alto nivel y tipado dinámico.
  • Sintaxis muy legible y limpia.
  • Soporta múltiples paradigmas: orientado a objetos, funcional, imperativo.
  • Cuenta con una enorme cantidad de librerías (NumPy, Pandas, TensorFlow, Flask, Django, etc.).

Tipos de datos:

1.Numericos

  • int: enteros (1, -5)
  • float: decimales (3.14)
  • complex: números complejos (3+2j)

2. Booleanos

  • bool: True o False

3.Texto

  • str: cadenas de texto ("Hola mundo")

4.Secuencias

  • list: listas mutables ([1, 2, 3])
  • tuple: tuplas inmutables ((1, 2, 3))
  • range: secuencias de números (range(5))

Ventajas

  • Fácil de aprender y leer.
  • Gran comunidad.
  • Portabilidad y flexibilidad.
  • Muy útil para prototipado rápido.

Desventajas

  • Más lento que lenguajes compilados.
  • Alto consumo de memoria.
  • No ideal para aplicaciones móviles.
