JavaScript

  • Características: Lenguaje interpretado, orientado a objetos, usado en el navegador.
  • Tipos de datos: Number, String, Boolean, Object, Array.
  • Ventajas: Interactividad, integración con HTML y CSS, ejecución rápida en el navegador.
  • Desventajas: Puede ser inseguro si se usa mal, comportamiento diferente entre navegadores.
