C#

Información General: Lenguaje desarrollado por Microsoft, ampliamente usado en aplicaciones Windows y videojuegos con Unity orientado a objetos

Características: Es similar a java pero esta mas orientado a objetos, ejecutado en .NET, y es de fácil integración con productos Microsoft.

Tipos de Datos: int, float, double, char, string, bool, object, dynamic.

Ventajas: Ideal para aplicaciones Windows, excelente para videojuegos en Unity.

Desventajas: Dependiente de .NET, menos flexible que otros lenguajes.
