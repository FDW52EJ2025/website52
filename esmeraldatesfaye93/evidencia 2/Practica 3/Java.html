Java

Información General: Lenguaje orientado a objetos, utilizado en aplicaciones empresariales, web y móviles. Compatible con cualquier plataforma que tenga la JVM y muy usado en empresas.

Características: Multiplataforma, robusto, seguro.

Tipos de Datos: byte, short, int, long, float, double, char, boolean.

Ventajas: Multiplataforma, Portabilidad, alto rendimiento, y una fuerte comunidad.

Desventajas:Consumo de memoria, código más verboso.
