JavaScript

Información General: Lenguaje de programación utilizado para el desarrollo web, principalmente para la creación de páginas dinámicas e interactivas.

Características: Interpretado, orientado a eventos, ejecutado en el navegador.

Tipos de Datos: Number, String, Boolean, Object, Array, Null, Undefined.

Ventajas: Esencial para la web, fácil de aprender, ejecutable en todos los navegadores.

Desventajas:Dependiente del navegador, puede ser inseguro y pues no se maneja adecuadamente.
