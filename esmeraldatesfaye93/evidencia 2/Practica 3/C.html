C

Información General: Lenguaje base de bajo nivel usado principalmente en sistemas operativos y software embebido.

Características: Compilado, estructurado, control directo sobre el hardware.

Tipos de Datos: int, float, double, char, void, enum.

Ventajas: Rápido, ideal para sistemas operativos y programación de hardware.

Desventajas: No orientado a objetos, manejo manual de memoria puede ser peligroso y poca seguridad.
