Python

Información General: Python es un lenguaje fácil de aprender y muy popular en ciencia de datos, desarrollo web e inteligencia artificial..

Características: Lenguaje interpretado, sintaxis clara, soporta múltiples paradigmas (como orientado a objetos y funcional).

Tipos de Datos: int, float, str, list, tuple, dict, set, bool.

Ventajas: Sintaxis sencilla, gran comunidad, extensa librería estándar.

Desventajas: Más lento que otros lenguajes compilados, alto consumo de memoria.
