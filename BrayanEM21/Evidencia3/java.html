Características

  • Orientado a objetos
  • Compilado e interpretado (JVM)
  • Multiplataforma

Tipos de Datos

  • int, float, double, char, boolean, String

Ventajas

  • Portabilidad entre sistemas
  • Gran comunidad y soporte
  • Uso empresarial y académico

Desventajas

  • Mayor consumo de recursos
  • Más verboso que otros lenguajes
