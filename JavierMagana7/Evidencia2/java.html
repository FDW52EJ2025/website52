Java

Descripción:

Java es un lenguaje de programación de alto nivel, orientado a objetos y de propósito general, desarrollado por Sun Microsystems en 1995 (ahora propiedad de Oracle). Su lema, "Write Once, Run Anywhere" (Escribe una vez, ejecuta en cualquier parte), refleja su principal ventaja: los programas Java se compilan en bytecode, que se ejecuta en la Java Virtual Machine (JVM), lo que permite que el mismo programa se ejecute en múltiples plataformas sin necesidad de recompilación. Java es ampliamente utilizado en desarrollo web, aplicaciones empresariales, dispositivos móviles (especialmente Android), y sistemas integrados.

Características Principales del Lenguaje:

* Portabilidad Multiplataforma
* Lenguaje Compilado e Interpretado
* Orientado a objetos
* Seguro y Robusto
* Multihilo (Multithreading)
* Gran Ecosistema de Bibliotecas y Frameworks

Enumeraciones (enum)

Ventajas de Java:

Portabilidad, orientado a objetos, gran comunidad y documentación, automatización de memoria, escalabilidad, alto nivel de seguridad, frameworks empresariales.

Desventajas de Java

Rendimiento inferior a lenguajes nativos, uso Intensivo de memoria, sintaxis verbosa, tiempo de arranque de la JVM, actualizaciones constantes.
