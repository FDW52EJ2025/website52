JavaScript

¿Qué es? Lenguaje de alto nivel, interpretado, principalmente usado en desarrollo web para crear interactividad.

Caracteristicas Principales

Ejecutado en el navegador.

No necesita compilación.

Usado tanto en frontend como en backend (con Node.js).

Ventajas

Muy versátil.

Amplia comunidad y bibliotecas.

Funciona en todos los navegadores.

Desventajas

Puede ser difícil de depurar.

Desempeño limitado en comparación con lenguajes compilados.

Tipos de datos

String, Number, Boolean, Array, Object, etc.
