Java

¿Qué es? Lenguaje orientado a objetos, ampliamente utilizado para aplicaciones empresariales, Android y desarrollo web.

Caracteristicas Principales

Portabilidad (Write Once, Run Anywhere - WORA).

Gestión automática de memoria.

Seguridad y robustez.

Ventajas

Plataforma independiente (JVM).

Buena documentación y comunidad activa.

Desventajas

Consumo de memoria alto.

No tan rápido como C++.

Tipos de datos

int, float, double, char, String, etc.
