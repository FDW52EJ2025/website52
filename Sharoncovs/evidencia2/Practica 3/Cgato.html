C#

¿Qué es? Lenguaje desarrollado por Microsoft, orientado a objetos, usado en desarrollo de aplicaciones web y videojuegos (Unity).

Caracteristicas Principales

Integración con .NET Framework.

Similar a Java.

Soporte para múltiples paradigmas.

Ventajas

Ideal para aplicaciones Windows.

Gran cantidad de herramientas.

Compatible con Unity.

Desventajas

Depende de plataformas Microsoft.

No tan portátil como Java.

Tipos de datos

int, float, double, char, string, etc.
