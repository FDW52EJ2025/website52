C: Lenguaje de Programación

Es uno de los más influyentes y fundamentales en la historia del desarrollo de software. Fue creado entre 1969 y 1972 por Dennis Ritchie en los Laboratorios Bell con el propósito inicial de reescribir el sistema operativo UNIX, permitiendo así un código más eficiente, portable y con acceso directo al hardware sin depender del lenguaje ensamblador

C se considera un lenguaje de nivel medio porque combina características de los lenguajes de bajo nivel (como manipulación de memoria y punteros) con estructuras típicas de lenguajes de alto nivel (como funciones y estructuras de control). Este equilibrio lo convierte en una herramienta poderosa tanto para el desarrollo de sistemas como para aplicaciones más generales

A lo largo del tiempo, C ha sido la base sobre la que se han desarrollado muchos otros lenguajes modernos, incluyendo C++, Java, C#, Objective-C, PHP y más. Incluso los compiladores e intérpretes de otros lenguajes suelen estar escritos en C.

Características principales

Característica Descripción
Lenguaje compilado El código se traduce directamente a lenguaje máquina, lo que lo hace muy rápido en su ejecución.
Procedimental Organiza el código en funciones y procedimientos, enfocándose en la lógica secuencial.
Control directo del hardware Permite manipular memoria y recursos a bajo nivel mediante punteros y direcciones.
Portabilidad Dependiendo del compilador, los programas pueden adaptarse a múltiples plataformas con mínimas modificaciones.
Eficiencia Ideal para sistemas donde se requiere rapidez y bajo consumo de recursos como sistemas embebidos o sistemas operativos.
Simplicidad en la sintaxis Su estructura es directa y lógica, aunque en proyectos grandes puede volverse compleja de gestionar.

Tipos de datos

Tipo de dato Ejemplo Descripción
Datos básicos
int int edad = 25; Entero de 32 bits, se usa comúnmente para contar o calcular
float float nota = 8.5f; Número decimal de precisión simple (32 bits), requiere sufijo f
double double precio = 99.99; Decimal de doble precisión (64 bits), mayor precisión que float
char char inicial = 'J'; Representa un solo carácter (usualmente 1 byte) de tipo ASCII o Unicode
boolean _Bool activo = 1; Solo puede contener 1 (verdadero) o 0 (falso); en C no hay tipo boolean como tal, se utiliza _Bool
byte char b = 100; En C no existe el tipo byte, pero se puede utilizar char para almacenar valores pequeños (8 bits)
short short s = 32000; Entero corto de 16 bits, entre -32,768 y 32,767
long long poblacion = 8000000L; Entero largo de 32 o 64 bits, requiere sufijo L
Datos complejos
String char nombre[] = "Ana"; En C no existe el tipo String, pero se utiliza un arreglo de char para representar cadenas de texto
Array int edades[] = {18, 25, 30}; Conjunto de elementos del mismo tipo, accesibles por índice
Datos definidos por el usuario
struct struct Persona { int edad; char nombre[20]; }; Define un tipo de dato compuesto que agrupa diferentes tipos de datos bajo un mismo nombre
enum enum Dia {LUNES, MARTES, MIERCOLES}; Define un conjunto de constantes con nombre, generalmente enteros
Pointer int *ptr = &edad; Variable que almacena la dirección de memoria de otra variable (puntero)

✅ Ventajas de C

  • Rápido y eficiente: Ideal para tareas que requieren rendimiento.
  • Portabilidad: Puede ejecutarse en distintas arquitecturas con pequeños cambios.
  • Control total del sistema: Manipulación directa de memoria, punteros y estructuras.
  • Lenguaje base: Muchos otros lenguajes están construidos sobre la sintaxis y filosofía de C.
  • Ideal para sistemas embebidos: Perfecto para aplicaciones en sistemas embebidos y firmware.

⚠️ Desventajas de C

  • No tiene manejo de objetos: No es orientado a objetos, lo que puede dificultar la modularidad.
  • Gestión manual de memoria: El programador debe liberar memoria, lo que puede causar errores como fugas.
  • Pocas herramientas de seguridad: No previene accesos indebidos o fallos como desbordamientos.
  • Curva de aprendizaje pronunciada: Especialmente al trabajar con punteros o estructuras complejas.
