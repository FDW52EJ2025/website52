JAVA

Java: Lenguaje de Programación

Es un lenguaje de programación de propósito general, orientado a objetos y compilado, desarrollado por Sun Microsystems en 1995 (ahora propiedad de Oracle). Su lema "escribe una vez, ejecuta en cualquier lugar" resume su gran ventaja: el código Java puede ejecutarse en cualquier dispositivo que tenga la Java Virtual Machine (JVM) instalada.

Java es ampliamente utilizado en aplicaciones empresariales, desarrollo móvil (especialmente Android), sistemas web y software de escritorio. Es uno de los lenguajes más enseñados y usados profesionalmente.

Características princiaples

Característica Descripción
Multiplataforma Gracias a la JVM, se puede ejecutar el mismo código en distintos sistemas operativos sin modificaciones.
Orientado a objetos Todo en Java está basado en clases y objetos, lo que mejora la organización, reutilización y mantenimiento del código.
Compilado e interpretado Se compila a bytecode y luego se ejecuta en la máquina virtual Java (JVM), lo que permite portabilidad y seguridad.
Fuertemente tipado Las variables deben tener un tipo definido, lo que ayuda a evitar errores durante la ejecución.
Gestión automática de memoria Incluye un recolector de basura que libera memoria no utilizada automáticamente.
Seguridad Proporciona mecanismos de control de acceso y protección de datos para evitar errores y amenazas.

Tipos de datos en Java

Tipo de dato Ejemplo Descripción
Datos básicos
int int edad = 25; Entero de 32 bits, se usa comúnmente para contar o calcular
float float nota = 8.5f; Número decimal de precisión simple (32 bits), requiere sufijo f
double double precio = 99.99; Decimal de doble precisión (64 bits), mayor precisión que float
char char inicial = 'J'; Representa un solo carácter unicode
boolean boolean activo = true; Solo puede contener true o false
byte byte b = 100; Entero pequeño de 8 bits, útil para ahorrar memoria
short short s = 32000; Entero corto de 16 bits, entre -32,768 y 32,767
long long poblacion = 8000000L; Entero largo de 64 bits, requiere sufijo L
Datos complejos
String String nombre = "Ana"; Cadena de texto, clase utilizada para manipular texto
Array int[] edades = {18, 25, 30}; Conjunto de elementos del mismo tipo
Datos definidos por el usuario
class class Persona { String nombre; int edad; } Define objetos con atributos y métodos; base de la POO
enum enum Dia {LUNES, MARTES, MIERCOLES}; Define un conjunto de constantes con nombre
Wrapper Integer edad = 30; Clases que encapsulan tipos primitivos (como int o double)

📈 Ventajas

  • Altamente portable: El código funciona en distintos sistemas operativos gracias a la JVM.
  • Orientado a objetos: Facilita la organización y reutilización del código en proyectos grandes.
  • Amplia comunidad: Existe mucha documentación, soporte, foros y herramientas disponibles.
  • Seguridad integrada: Ofrece mecanismos que previenen errores comunes y vulnerabilidades.
  • Automatización de memoria: El recolector de basura gestiona la memoria sin intervención del programador.
  • Base de Android: Es el lenguaje principal para crear aplicaciones móviles en Android.

📉 Desventajas

  • Verboso: Se necesita escribir más líneas de código para tareas simples en comparación con otros lenguajes.
  • Alto consumo de memoria: La JVM requiere recursos que pueden ser excesivos para sistemas limitados.
  • Menor rendimiento que C++: Aunque es rápido, no alcanza el nivel de eficiencia de lenguajes de bajo nivel.
  • Requiere configuración: Es necesario instalar y configurar herramientas como JDK y JVM para poder ejecutar programas.
