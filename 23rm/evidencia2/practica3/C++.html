C++: Lenguaje de Programación

Es un lenguaje de programación de propósito general, compilado y de alto rendimiento. Fue creado por Bjarne Stroustrup a principios de los años 80 como una extensión del lenguaje C, añadiendo características de programación orientada a objetos.
C++ es ampliamente utilizado en el desarrollo de software de sistemas, videojuegos, aplicaciones en tiempo real, sistemas embebidos y más. Es un lenguaje poderoso, pero con una curva de aprendizaje más exigente.

Características princiaples

Característica Descripción
Compilado Requiere ser traducido a lenguaje máquina antes de ejecutarse, lo que permite alto rendimiento.
Orientado a objetos Permite organizar el código usando clases, objetos, herencia y encapsulamiento.
Alto control de hardware Ofrece control detallado de la memoria, punteros y recursos del sistema.
Multiplataforma Se puede ejecutar en diferentes sistemas operativos con mínimos ajustes.
Lenguaje híbrido Combina programación estructurada (como C) y orientada a objetos.
Estándar industrial Se usa en sistemas operativos, motores de videojuegos y software de misión crítica.

Tipos de datos en C++

Tipo de dato Ejemplo Descripción
Datos básicos
int 5 Entero (positivo o negativo)
float 3.14 Números decimales
str (en C++ moderno) "Hola" Cadenas de texto
bool True, False Valores booleanos
char "A", 9 Un solo carácter
double 3.1415926 Similar a float pero más preciso
Datos complejos
array int nums[5] = {1,2,3,4,5}; Conjunto de elementos del mismo tipo
Datos definidos por el usuario
struct struct Persona {string, nombre:int edad; }; Estructura definida por el usuario para agrupar datos
class class Coche {public: string modelo; int año; }; Define objetos con propiedades (atributos) y funciones (métodos)
enum enum Dia {Lunes, Martes, Miercoles}; Conjunto de constantes enteras con nombre

📈 Ventajas

  • Alto rendimiento: Muy eficiente en el uso de recursos y rápido en ejecución.
  • Control total sobre memoria: Ideal para desarrollo de sistemas y drivers.
  • Orientado a objetos: Mejora la organización y mantenimiento del código.
  • Portabilidad: Se puede compilar para múltiples plataformas.
  • Amplio uso profesional: Usado en la industria de videojuegos, sistemas operativos, motores de simulación, etc.

📉 Desventajas

  • Complejidad alta: Es un lenguaje difícil para principiantes.
  • Gestión manual de memoria: Puede causar errores graves si no se maneja correctamente (como fugas de memoria).
  • Compilación lenta: Requiere tiempo y configuración para compilar grandes proyectos.
  • Menor productividad: Comparado con lenguajes como Python o JavaScript, lleva más tiempo escribir y probar código.
