Java

Caracteristicas Principales

  • Orientado a objetos: Java organiza el código en clases y objetos, lo que promueve la reutilización y la modularidad del código
  • Sencillo: A pesar de ser un lenguaje potente, Java tiene una sintaxis bastante clara y sencilla, lo que facilita su aprendizaje y uso.
  • Seguridad: Java tiene características de seguridad integradas, como la verificación de bytecode en la JVM y el modelo de seguridad basado en permisos.

Tipo de datos

  • Objetos: Cualquier instancia de una clase.
  • Arrays: Estructuras de datos que almacenan múltiples elementos del mismo tipo.
  • int: 4 bytes, valores entre -2,147,483,648 y 2,147,483,647.
  • byte: 1 byte, valores entre -128 y 127.
  • double: 8 bytes, para números decimales con precisión doble.
  • short: 2 bytes, valores entre -32,768 y 32,767.
Ventajas Desventajas
- Portabilidad (Ejecuta en cualquier plataforma con JVM).
- Orientado a objetos (Mejora la modularidad y reutilización).
- Multithreading (Permite ejecución simultánea de procesos).
- Seguridad (Verificación de bytecode, control de acceso).
- Amplia biblioteca estándar (Java API). 		- Requiere JVM (Puede ser más lento).
- Complejidad para principiantes.
- Consumo de memoria relativamente alto.
- Requiere gestión adecuada de hilos (Multithreading).
