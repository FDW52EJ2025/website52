C#

Caracteristicas Principales

  • Lenguaje moderno orientado a objetos, desarrollado por Microsoft.
  • Compilado a lenguaje intermedio (MSIL), corre en la plataforma .NET.
  • Muy usado en desarrollo de aplicaciones de escritorio, web y videojuegos (especialmente con Unity).

Tipo de datos

  • Primitivos: int, float, double, char, bool
  • Avanzados: string, array, List<>, Dictionary<>, class, struct, enum
Ventajas Desventajas
- Lenguaje moderno, limpio y potente.
- Ideal para apps Windows y videojuegos (Unity).
- Buen soporte de herramientas (Visual Studio, .NET).
- Gestión automática de memoria. 		- Muy dependiente del ecosistema Microsoft.
- Más pesado que otros lenguajes para apps simples.
- Menos soporte fuera de Windows (aunque ha mejorado).
