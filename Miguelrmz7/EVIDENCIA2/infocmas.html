C++

Caracteristicas

  • Lenguaje compilado, de medio-bajo nivel.
  • Permite programación orientada a objetos, estructurada y genérica.
  • Muy eficiente en el uso de memoria y rendimiento.

Tipo de datos

  • Primitivos: int, float, double, char, bool
  • Compuestos: struct, array, pointer, class
  • STL: vector, map, set, etc.
Ventajas Desventajas
- Muy rápido y eficiente.
- Control total de memoria.
- Usado en videojuegos, sistemas operativos y software de alto rendimiento. 		- Sintaxis compleja.
- Gestión manual de memoria (propenso a errores).
- Curva de aprendizaje empinada.
