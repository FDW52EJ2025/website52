Python

Python es un lenguaje de programación de alto nivel, interpretado y de propósito general, conocido por su sintaxis clara y legible, lo que lo hace ideal tanto para principiantes como para desarrolladores experimentados. Fue creado por Guido van Rossum y su primera versión fue lanzada en 1991.Python es multiparadigma, lo que significa que permite programar de forma imperativa, orientada a objetos e incluso funcional. Además, cuenta con una amplia colección de bibliotecas y frameworks que lo hacen útil en diversos campos

C++

C++ es un lenguaje de programación compilado, de propósito general y orientado a objetos, basado en el lenguaje C. Es ampliamente utilizado para el desarrollo de software de sistemas, videojuegos y aplicaciones de alto rendimiento, gracias a su eficiencia y control sobre los recursos del sistema.

C#

C# (C Sharp) es un lenguaje de programación moderno, orientado a objetos y desarrollado por Microsoft como parte de su plataforma .NET. Es utilizado para crear aplicaciones de escritorio, web, móviles y videojuegos (especialmente con Unity), destacando por su versatilidad y facilidad de uso en entornos Windows.

Javascript

JavaScript es un lenguaje de programación interpretado y orientado a objetos, utilizado principalmente para crear interactividad en páginas web. Se ejecuta en el navegador y permite desarrollar desde animaciones hasta aplicaciones web completas, tanto en el frontend como en el backend (con Node.js).

Java

Java es un lenguaje de programación de alto nivel, orientado a objetos, ampliamente utilizado para el desarrollo de aplicaciones de software en diferentes plataformas. Fue creado por Sun Microsystems (ahora propiedad de Oracle) en 1995. Es conocido por su lema "escribe una vez, ejecuta en cualquier lugar", lo que significa que las aplicaciones Java pueden ejecutarse en cualquier dispositivo que tenga una Máquina Virtual de Java (JVM).
