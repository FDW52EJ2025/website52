¿Como funciona el lenguaje Javascript

Desde el punto de vista del desarrollo Front End, un sitio web consta de tres capas:La primera, escrita con HTML (Hypertext MarkupLanguage), describe la estructura básica de la página y permite ordenar el contenido según títulos, encabezados, párrafos, etc. Luego vienen las hojas de estilo en CSS (Cascading Style Sheets), que determinan el diseño y la presentación del sitio web. JavaScript forma la última capa que le agrega emoción y movimiento al sitio web, y se utiliza para transformar una página estática en una interfaz dinámica. La ejecución de los scripts o bloques de código de JavaScript se basa en eventos, por ejemplo, clics con el mouse o entradas del usuario.

Caracteristicas

      1.-Lenguaje Interpretado: JavaScript es un lenguaje interpretado, lo que significa que el código se ejecuta directamente en el navegador sin necesidad de compilación previa.
      2.-Orientado a Objetos: Aunque no es un lenguaje puramente orientado a objetos, JavaScript permite la creación y manipulación de objetos, y soporta la programación basada en prototipos.
      3.-Dinámico: JavaScript es un lenguaje dinámico, lo que significa que los tipos de datos se determinan en tiempo de ejecución y las variables pueden cambiar de tipo.

Tipos de datos

number, string, boolean, object, array, null, undefined

Ventajas

      1.-Muy versátil (frontend y backend con Node.js).
      2.-Interactivo y moderno.

Desventajas

      1.-Puede ser difícil de depurar.
      2.-Sintaxis flexible que puede causar errores.
