Lenguaje C

¿Como funciona el lenguaje C?

El lenguaje C funciona como un lenguaje de programación de bajo nivel que permite controlar directamente el hardware de la computadora su código es compilado lo que significa que se traduce a lenguaje máquina antes de ejecutarse es muy eficiente en cuanto a velocidad y uso de memoria permite crear programas rápidos y ligeros es estructurado lo que significa que se organiza en funciones y bloques de código su sintaxis es precisa y se basa en instrucciones claras y directas ofrece control total sobre variables memoria punteros y estructuras se usa ampliamente en sistemas operativos software de sistemas y programación embebida requiere declarar el tipo de dato de cada variable antes de usarla y permite manipular memoria directamente a través de punteros

Caracteristicas

      1.-La característica clave del lenguaje de programación C es su minimalismo: fue pensado como herramienta sencilla para escribir programas que se compilan con un compilador, es decir, el código original se convierte en un archivo que la máquina puede leer directamente, sin aplicaciones adicionales.
      2.-El compilador es de un solo paso, y la biblioteca estándar es bastante limitada. La combinación de estos factores hacen que los compiladores C sean bastante fáciles de implementar, así que los programas escritos en C se pueden compilar en cualquier computadora.
      3.-El lenguaje C es un lenguaje de nivel medio bajo, así que no requiere de muchos recursos pero ofrece una buena respuesta y una ejecución rápida de programas.

Tipos de datos

int, float, char, double, arrays, pointers

Ventajas

      1.-Extremadamente rápido.
      2.-Perfecto para sistemas operativos y drivers.

Desventajas

      1.-No tiene orientación a objetos.
      2.-Curva de aprendizaje para principiantes.
