Java

¿Como funciona el lenguaje Java?

Los programas Java se compilan en bytecode, que es un lenguaje de bajo nivel que puede ser ejecutado por la Máquina Virtual Java (JVM). Esto permite ejecutar programas Java en cualquier ordenador que tenga instalada una JVM, independientemente de su sistema operativo

Caracteristicas

      1.-Portabilidad: Java es independiente de la plataforma, lo que significa que el código puede ejecutarse en cualquier dispositivo que tenga la JVM (Java Virtual Machine) instalada.
      2.-Orientación a objetos: Java sigue el paradigma de la programación orientada a objetos, lo que permite la encapsulación, herencia y polimorfismo.
      Seguridad: Java fue diseñado con un fuerte enfoque en la seguridad, lo que lo hace adecuado para aplicaciones en red.

Tipos de datos

int, float, double, char, boolean, String, arrays, objetos

Ventajas

      1.-Bueno para desarrollo empresarial y Android.
      2.-Portabilidad entre sistemas.

Desventajas

      1.-Más lento que C++.
      2.-Más código para cosas simples.
