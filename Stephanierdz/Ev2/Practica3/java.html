Características Principales

Java es un lenguaje de programación de propósito general, orientado a objetos y diseñado para tener la menor cantidad de dependencias de implementación.

Tipos de Datos

Java soporta tipos de datos primitivos como enteros, flotantes, booleanos y caracteres.

Ventajas

  • Portabilidad gracias a la máquina virtual de Java (JVM).
  • Gran comunidad y soporte.
  • Seguridad y robustez en la gestión de memoria.

Desventajas

  • Rendimiento inferior en comparación con lenguajes compilados como C++.
  • Consumo de memoria elevado.

Ejemplo "Hola Mundo"

public class HolaMundo {
    public static void main(String[] args) {
        System.out.println("Hola Mundo");
