Lenguaje de programación JavaScript

¿Qué es JavaScript?

JavaScript es un lenguaje de programación interpretado, orientado a objetos, que se utiliza principalmente en el desarrollo web para crear páginas dinámicas e interactivas. Fue desarrollado originalmente por Netscape en 1995 y es uno de los pilares del desarrollo frontend junto con HTML y CSS.

Características principales

  • Interpretado: No necesita compilación previa.
  • Basado en eventos: Permite reaccionar a acciones del usuario.
  • Multiplataforma: Funciona en cualquier navegador moderno.
  • Lenguaje dinámico: Permite cambiar tipos de datos durante la ejecución.

Tipos de datos comunes

  • Number: números (enteros o decimales)
  • String: cadenas de texto
  • Boolean: verdadero o falso
  • Object: colecciones de pares clave-valor
  • Array: listas ordenadas
  • Null y undefined: valores vacíos o no definidos

Ventajas

  • Ejecuta directamente en el navegador.
  • Ideal para crear interfaces interactivas.
  • Compatible con múltiples plataformas.
  • Gran ecosistema con muchas librerías y frameworks (como React, Vue, Angular).

Desventajas

  • Menor seguridad, ya que el código es visible desde el navegador.
  • El manejo de errores puede ser menos estructurado que en otros lenguajes.
  • Comportamientos inconsistentes entre distintos navegadores (aunque hoy menos común).

Ejemplo: Hola Mundo en JavaScript

<script>
    console.log("Hola, mundo");
</script>

Este fragmento muestra Hola, mundo en la consola del navegador. Se usa console.log() para imprimir mensajes.
