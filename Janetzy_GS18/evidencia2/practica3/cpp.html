Lenguajes de programación

¿Qué es C++?

C++ es un lenguaje de programación de propósito general que fue creado por Bjarne Stroustrup a principios de los años 80 como una extensión del lenguaje C. Introduce características de programación orientada a objetos, manteniendo la eficiencia y flexibilidad del C original.

Características principales

  • Orientado a objetos: Permite usar clases, herencia, encapsulamiento y polimorfismo.
  • Lenguaje compilado: Convierte el código fuente en código máquina para una ejecución rápida.
  • Soporte de programación genérica: Permite crear funciones y clases reutilizables con plantillas.
  • Compatible con C: Se puede usar código C dentro de programas en C++.

Ventajas

  • Alto rendimiento en ejecución.
  • Muy flexible para trabajar tanto a bajo como a alto nivel.
  • Amplio soporte para estructuras de datos y algoritmos complejos.
  • Gran comunidad y abundantes recursos de aprendizaje.

Desventajas

  • Puede ser más complejo de aprender para principiantes.
  • No tiene recolección automática de basura como Java o C#.
  • Errores de memoria pueden ser difíciles de depurar.

Ejemplo: Hola Mundo en C++

#include <iostream>
        
        int main() {
        std::cout << "Hola, mundo" << std::endl;
        return 0;
        }

Este programa muestra Hola, mundo. #include <iostream> permite usar entrada y salida estándar. std::cout imprime texto en pantalla. return 0; indica que el programa terminó correctamente.
