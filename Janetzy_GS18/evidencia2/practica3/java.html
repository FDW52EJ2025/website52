Lenguajes de programación

¿Qué es Java?

Java es un lenguaje de programación orientado a objetos, desarrollado por Sun Microsystems en 1995 (hoy propiedad de Oracle). Es conocido por su portabilidad entre plataformas gracias a su máquina virtual, la Java Virtual Machine (JVM), que permite ejecutar el mismo programa en diferentes sistemas operativos sin modificaciones.

Características principales

  • Orientado a objetos: Todo en Java está basado en clases y objetos.
  • Portabilidad: "Escribe una vez, ejecuta en cualquier lugar" gracias a la JVM.
  • Lenguaje seguro: Maneja errores y excepciones, y no permite acceso directo a memoria.
  • Recolección de basura: Libera automáticamente la memoria que ya no se usa.

Ventajas

  • Multiplataforma.
  • Gran comunidad y amplia documentación.
  • Ideal para desarrollo empresarial, móvil (Android) y web.
  • Automatiza gestión de memoria.

Desventajas

  • Puede ser más lento que lenguajes compilados directamente como C++.
  • Consumo de memoria relativamente alto.
  • Sintaxis más verbosa comparada con lenguajes como Python.

Ejemplo: Hola Mundo en Java

public class HolaMundo {
        public static void main(String[] args) {
            System.out.println("Hola, mundo");
        }
        }

Este programa imprime Hola, mundo en pantalla. public static void main(String[] args) es el método principal desde donde se ejecuta el programa. System.out.println() muestra el mensaje.
