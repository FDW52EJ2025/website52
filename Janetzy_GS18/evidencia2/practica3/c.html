Lenguajes de programación

¿Qué es C?

El lenguaje C es un lenguaje de programación de propósito general, estructurado y compilado, creado a principios de los años 70 por Dennis Ritchie en los laboratorios Bell. Es uno de los lenguajes más influyentes de la historia, y muchas tecnologías modernas, sistemas operativos y lenguajes (como C++, C#, Java) están basados o inspirados en él.

Características principales

  • Lenguaje de bajo nivel: Permite trabajar directamente con memoria y hardware.
  • Compilado: El código fuente se transforma en código máquina antes de ejecutarse.
  • Eficiente: Altamente veloz y con gran control sobre el rendimiento del sistema.
  • Portabilidad: Los programas en C pueden compilarse en distintos sistemas con mínimas modificaciones.

Ventajas

  • Alto rendimiento y velocidad de ejecución.
  • Permite un control preciso del sistema y de la memoria.
  • Base para muchos otros lenguajes modernos.
  • Ideal para desarrollar sistemas operativos, controladores y software embebido.

Desventajas

  • No tiene manejo automático de memoria (como Python o Java).
  • No posee características modernas como programación orientada a objetos.
  • La sintaxis puede ser más compleja para principiantes.

Ejemplo: Hola Mundo en C

#include <stdio.h>
        
        int main() {
        printf("Hola, mundo\n");
        return 0;
        }

Este código imprime Hola, mundo. #include <stdio.h> importa la biblioteca estándar de entrada/salida. printf() muestra el texto, y return 0; indica que el programa terminó correctamente.
