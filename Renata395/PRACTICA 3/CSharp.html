LENGUAJE C#

C# o tambien llamado CSharp es un lenguaje de programaciónmoderno, innovador, de código abierto, multiplataforma orientado a objetos y uno de los 5 principales lenguajes de programación de GitHub. Fue desarrollado por Microsoft y es uno de los lenguajes más populares para la plataforma .NET. C# es conocido por ser un lenguaje orientado a objetos, lo que significa que está diseñado para facilitar la creación de software modular y reutilizable.

- Características Principales

C# es orientado a componentes, tiene recolección de basura automática y seguridad de tipos. Además, C# ofrece instrucciones seguras, una unificación de tipos y la capacidad de extender operadores y modificadores.

  • Permite definir tipos y métodos genéricos
  • Admite técnicas orientadas a objetos como herencia y polimorfismo
  • Incorpora mecanismos para asegurar que los accesos a tipos de datos se lleven a cabo de forma correcta

- Tipos de Datos

C# admite nueve tipos enteros:

  • sbyte
  • byte
  • short
  • ushort
  • int
  • uint
  • long
  • ulong
  • char

- Ventajas

  • Facilidad de aprendizaje que es un lenguaje de alto nivel y de fácil aprendizaje
  • Puedes escribir programas en C# en un entorno tan simple como un bloc de texto.

- Desventajas

  • No se puede heredar el código y las clases escritas en Java
  • Más dependiente de Microsoft Windows
