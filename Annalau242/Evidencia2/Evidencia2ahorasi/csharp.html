C#

Información general

C# es un lenguaje de programación desarrollado por Microsoft en el año 2000. Se utiliza principalmente en desarrollo de aplicaciones de escritorio, videojuegos con Unity y aplicaciones web con .NET.

Características principales

  • Paradigma: Orientado a objetos
  • Tipado: Estático y fuerte
  • Integración con la plataforma .NET

Tipos de datos

  • int: Enteros
  • double: Números decimales
  • char: Caracteres
  • bool: Valores booleanos
  • string: Cadenas de texto

Ventajas y desventajas

Ventajas

  • Interoperabilidad con .NET
  • Buena gestión de memoria con el Garbage Collector
  • Sintaxis moderna y sencilla

Desventajas

  • Dependencia de .NET
  • No es tan rápido como C++

Ejemplo de "Hola Mundo"


using System;

class Program {
    static void Main() {
        Console.WriteLine("Hola Mundo");
    }
}
