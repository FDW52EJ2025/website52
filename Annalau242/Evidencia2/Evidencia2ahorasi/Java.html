Java

Información general

Java es un lenguaje de programación orientado a objetos desarrollado por Sun Microsystems en 1995. Es utilizado ampliamente en aplicaciones empresariales, móviles y de escritorio.

Características principales

  • Paradigma: Orientado a objetos
  • Tipado: Estático y fuerte
  • Portabilidad gracias a la JVM

Tipos de datos

  • int: Enteros
  • double: Números decimales
  • char: Caracteres
  • boolean: Valores booleanos
  • String: Cadenas de texto

Ventajas y desventajas

Ventajas

  • Multiplataforma
  • Gran comunidad y soporte
  • Manejo automático de memoria

Desventajas

  • Consumo alto de memoria
  • Menos eficiente que C++ en rendimiento

public class HolaMundo {
    public static void main(String[] args) {
        System.out.println("Hola Mundo");
