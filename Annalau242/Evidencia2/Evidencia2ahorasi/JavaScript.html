JavaScript

Información general

JavaScript es un lenguaje de programación interpretado, utilizado principalmente en el desarrollo web para hacer páginas dinámicas e interactivas.

Características principales

  • Paradigma: Multiparadigma (imperativo, orientado a objetos y funcional)
  • Tipado: Débil y dinámico
  • Ejecutado en el navegador

Tipos de datos

  • Number: Números
  • String: Cadenas de texto
  • Boolean: Valores booleanos
  • Object: Objetos
  • Undefined: Valor no definido

Ventajas y desventajas

Ventajas

  • Funciona en todos los navegadores
  • No requiere compilación
  • Amplia comunidad y soporte

Desventajas

  • Seguridad limitada
  • Tipado débil puede generar errores
