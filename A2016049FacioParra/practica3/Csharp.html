C#

Información General

C# es un lenguaje de programación desarrollado por Microsoft, ampliamente utilizado para aplicaciones de escritorio, videojuegos (con Unity) y desarrollo web con .NET.

Características Principales

  • Lenguaje orientado a objetos
  • Integración con el ecosistema .NET
  • Tipado estático y fuerte

Tipos de Datos

  • int, float, double, char, bool
  • arrays, listas, diccionarios

Ventajas

  • Gran soporte de Microsoft
  • Buena integración con aplicaciones empresariales
  • Uso en desarrollo de videojuegos con Unity

Desventajas

  • Dependencia de la plataforma .NET
  • Menos flexible que otros lenguajes como C++

Ejemplo Hola Mundo

            
using System;

class Program {
    static void Main() {
        Console.WriteLine("Hola, mundo!");
    }
}
