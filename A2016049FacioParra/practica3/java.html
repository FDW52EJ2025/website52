Java

Información General

Java es un lenguaje de programación orientado a objetos, multiplataforma y ampliamente utilizado en aplicaciones empresariales, móviles y de backend.

Características Principales

  • Lenguaje compilado e interpretado (JVM)
  • Orientado a objetos
  • Portabilidad gracias a la Máquina Virtual de Java (JVM)

Tipos de Datos

  • int, double, float, char, boolean
  • Arrays, Strings, Clases, Interfaces

Ventajas

  • Alto nivel de seguridad
  • Portabilidad en distintas plataformas
  • Amplia comunidad y soporte

Desventajas

  • Consumo elevado de memoria
  • Rendimiento inferior a lenguajes compilados como C++

public class HolaMundo {
    public static void main(String[] args) {
        System.out.println("Hola, mundo!");
