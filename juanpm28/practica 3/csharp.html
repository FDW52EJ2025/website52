C#

Características Principales

  • Orientado a objetos moderno
  • Desarrollado por Microsoft
  • Parte de la plataforma .NET
  • Tipado estático fuerte
  • Recolección automática de basura
  • Soporte para programación asíncrona
  • LINQ (Language Integrated Query)
  • Interoperabilidad con otros lenguajes .NET

Tipos de Datos

  • Tipos de valor:
    • bool
    • byte, sbyte
    • short, ushort
    • int, uint
    • long, ulong
    • float
    • double
    • decimal
    • char
    • struct
    • enum
  • Tipos de referencia:
    • class
    • interface
    • delegate
    • string
    • array
    • object

Ventajas

  • Excelente para desarrollo de aplicaciones Windows
  • Gran integración con el ecosistema Microsoft
  • Sintaxis moderna y expresiva
  • Amplio conjunto de bibliotecas (.NET Framework)
  • Bueno para desarrollo web (ASP.NET)
  • Ideal para desarrollo de juegos (Unity)
  • Soporte multiplataforma (.NET Core)
  • Excelentes herramientas de desarrollo (Visual Studio)

Desventajas

  • Curva de aprendizaje inicial
  • Menor portabilidad comparada con Java
  • Tradicionalmente limitado a Windows (aunque esto ha cambiado con .NET Core)
  • Puede ser más costoso (licencias de Visual Studio)
  • Mayor consumo de recursos comparado con C++
