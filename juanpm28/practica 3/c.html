C

Características Principales

  • Lenguaje de programación estructurado
  • Lenguaje de bajo nivel con sintaxis de alto nivel
  • Eficiente y rápido
  • Manejo directo de memoria
  • Portabilidad entre plataformas
  • Base para muchos otros lenguajes
  • Ideal para programación de sistemas

Tipos de Datos

  • Básicos:
    • char (caracteres)
    • int (enteros)
    • float (punto flotante)
    • double (punto flotante doble precisión)
    • void (sin tipo)
  • Modificadores:
    • short
    • long
    • signed
    • unsigned
  • Derivados:
    • Arrays
    • Punteros
    • Estructuras
    • Uniones

Ventajas

  • Alto rendimiento y eficiencia
  • Control preciso sobre el hardware
  • Excelente para programación de sistemas
  • Código muy portable
  • Biblioteca estándar pequeña pero eficiente
  • Base fundamental para aprender otros lenguajes
  • Ampliamente utilizado en sistemas embebidos

Desventajas

  • No tiene soporte directo para programación orientada a objetos
  • Gestión manual de memoria
  • Propenso a errores de punteros
  • No tiene recolección de basura
  • Depuración puede ser compleja
  • Falta de características modernas

Hola Mundo en C

#include <stdio.h>

int main() {
    printf("¡Hola, Mundo!\n");
    return 0;
}

Áreas de Aplicación

  • Sistemas Operativos
  • Controladores de dispositivos
  • Sistemas embebidos
  • Aplicaciones en tiempo real
  • Desarrollo de compiladores
  • Software de bajo nivel

Bibliotecas Estándar Importantes

  • stdio.h - Entrada/Salida estándar
  • stdlib.h - Funciones de utilidad general
  • string.h - Manipulación de cadenas
  • math.h - Funciones matemáticas
  • time.h - Funciones de tiempo
  • ctype.h - Clasificación de caracteres
