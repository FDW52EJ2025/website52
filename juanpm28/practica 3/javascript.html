JavaScript

Características Principales

  • Lenguaje interpretado
  • Dinámicamente tipado
  • Orientado a objetos basado en prototipos
  • Funciones de primera clase
  • Ejecución en el navegador
  • Soporte para programación asíncrona
  • Event-driven (basado en eventos)
  • Multiplataforma

Tipos de Datos

  • Primitivos:
    • number (números enteros y decimales)
    • string (cadenas de texto)
    • boolean (true/false)
    • undefined
    • null
    • symbol (ES6+)
    • bigint (ES2020+)
  • Objetos:
    • Object
    • Array
    • Function
    • Date
    • RegExp
    • Map y Set
    • Promise

Ventajas

  • Lenguaje universal para desarrollo web
  • Fácil de aprender
  • Gran comunidad y ecosistema
  • Amplia variedad de frameworks y bibliotecas
  • Desarrollo tanto frontend como backend (Node.js)
  • Actualizaciones regulares (ECMAScript)
  • Flexibilidad y versatilidad
  • Rico en APIs del navegador

Desventajas

  • Tipado dinámico puede llevar a errores
  • Diferentes comportamientos entre navegadores
  • Problemas de seguridad si no se maneja correctamente
  • Código puede ser visible al usuario
  • Precisión limitada con números decimales
  • Demasiadas formas de hacer lo mismo

Hola Mundo en JavaScript

// En el navegador
console.log("¡Hola, Mundo!");

// En HTML
<script>
    document.write("¡Hola, Mundo!");
</script>

// Alerta
alert("¡Hola, Mundo!");
