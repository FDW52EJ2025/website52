C++

Características Principales

  • Lenguaje compilado
  • Orientado a objetos
  • Tipado estático
  • Soporte para programación de bajo nivel
  • Alta eficiencia y rendimiento
  • Compatible con código C

Tipos de Datos

  • Enteros (int, short, long, unsigned)
  • Punto flotante (float, double)
  • Caracteres (char)
  • Booleanos (bool)
  • Arrays
  • Punteros
  • Referencias
  • Estructuras y Clases

Ventajas

  • Alto rendimiento y eficiencia
  • Control preciso sobre el hardware
  • Amplia biblioteca estándar
  • Portabilidad
  • Ideal para desarrollo de sistemas y aplicaciones complejas
  • Gran comunidad y recursos disponibles

Desventajas

  • Curva de aprendizaje pronunciada
  • Sintaxis compleja
  • Gestión manual de memoria
  • Posibilidad de errores de punteros
  • Tiempo de compilación largo en proyectos grandes

Hola Mundo en C++

#include <iostream>

int main() {
    std::cout << "¡Hola, Mundo!" << std::endl;
    return 0;
}

Ejemplo Alternativo con Namespace

#include <iostream>
using namespace std;

int main() {
    cout << "¡Hola, Mundo!" << endl;
    return 0;
}
