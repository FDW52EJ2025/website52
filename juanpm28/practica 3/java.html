Java

Características Principales

  • Orientado a objetos puro
  • Independiente de la plataforma (WORA - Write Once, Run Anywhere)
  • Tipado estático fuerte
  • Recolección automática de basura (Garbage Collection)
  • Multihilo
  • Seguridad integrada
  • Gran ecosistema de bibliotecas

Tipos de Datos

  • Primitivos:
    • byte
    • short
    • int
    • long
    • float
    • double
    • boolean
    • char
  • Referencia:
    • Clases
    • Interfaces
    • Arrays
    • Enumeraciones

Ventajas

  • Portabilidad excepcional
  • Gran comunidad y soporte
  • Amplia biblioteca estándar (JDK)
  • Excelente para aplicaciones empresariales
  • Gestión automática de memoria
  • Robusto y seguro
  • Ideal para desarrollo Android

Desventajas

  • Consumo de memoria relativamente alto
  • Velocidad de ejecución menor comparada con lenguajes compilados
  • Sintaxis más verbosa que otros lenguajes
  • Curva de aprendizaje inicial pronunciada
  • Requiere JVM para ejecutarse
