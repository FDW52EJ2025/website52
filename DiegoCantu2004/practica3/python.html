Python

Características Principales

  • Interpretado: Ejecuta el código línea por línea.

  • Fácil de usar: Tiene una sintaxis clara y es interactivo.

  • De código abierto: Es de código abierto y tiene fuentes incorporadas.

  • Orientado a objetos: Soporta la orientación a objetos.

  • Multiplataforma: Se puede ejecutar en diferentes sistemas operativos.

Tipos de datos

  1. Numéricos: int, float, complex

  2. Cadenas: str

  3. Secuencias: list, tuple, range

  4. Binarios: bytes, bytearray

  5. Diccionarios: dict

  6. Booleanos: bool

  7. Conjuntos: set, frozenset

Ventajas y desventajas

  • Ventajas:

  • Es legible y de fácil aprendizaje

  • Es versátil y se utiliza en muchos campos

  • Tiene un amplio ecosistema de bibliotecas y frameworks

  • Es gratuito y de código abierto

  • Desventajas:

  • Puede ser más lento que lenguajes compilados como C++ o Java

  • Puede consumir más memoria que otros lenguajes

  • Puede tener limitaciones en la representación de valores y resultados estadísticos

  • Puede presentar errores que solo se notan durante la ejecución
