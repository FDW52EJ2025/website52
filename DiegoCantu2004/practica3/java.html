Características Principales

  • Portable: El código Java compilado puede ejecutarse en cualquier plataforma que admita Java sin necesidad de recopilación.

  • Seguro: Utiliza un modelo de seguridad robusto que protege los sistemas contra posibles amenazas y vulnerabilidades.

  • Multihilo: Permite llevar a cabo varias tareas simultáneamente dentro del mismo programa.

  • Recolector de basura: Borra los objetos que no tienen referencias localizadas, liberando así la memoria que ocupaban.
