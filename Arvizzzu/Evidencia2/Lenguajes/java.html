Java

Características Principales

Java es un lenguaje de programación de propósito general, basado en clases, que es ampliamente utilizado para aplicaciones móviles, de escritorio y web. Es conocido por ser "write once, run anywhere" (escribe una vez, ejecuta en cualquier lugar).

Tipos de Datos

Java tiene tipos de datos primitivos como enteros, flotantes, dobles, caracteres, y booleanos, y también soporta clases y objetos.

Ventajas y Desventajas

Ventajas: Portabilidad, seguridad, y robustez.

Desventajas: Mayor consumo de recursos y rendimiento comparado con lenguajes como C++.

Ejemplo "Hola Mundo"

public class Main {
    public static void main(String[] args) {
        System.out.println("Hola Mundo");
