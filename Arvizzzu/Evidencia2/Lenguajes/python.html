Python

Python es un lenguaje de programación interpretado, de alto nivel y multipropósito. Es conocido por su sintaxis clara y legible.

Características

  • Sintaxis simple
  • Interpretado
  • Multiparadigma

Tipos de datos

int, float, str, list, tuple, dict, bool

Ventajas

  • Fácil de aprender
  • Gran comunidad
  • Multiplataforma

Desventajas

  • Menor velocidad que otros lenguajes compilados
