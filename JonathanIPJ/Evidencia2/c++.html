¿Que es C++?

C++ es un lenguaje de programación multiparadigma (orientado a objetos, procedural y genérico), de alto rendimiento y bajo nivel, ampliamente usado en sistemas críticos.

Caracteristicas de C++

✅Eficiencia y Rendimiento: Código compilado directamente a lenguaje máquina.

✅Orientado a Objetos (POO): Soporta clases, herencia, polimorfismo y encapsulamiento.

✅ Manipulación de Memoria Directa: Uso de punteros y gestión manual de memoria.

✅ Multiparadigma: Soporta programación estructurada, genérica (templates) y funcional.

Tipos de Datos

Tipo de Datos Tamaño Ejemplo

Ventajas de C++

Ventajas de C++

✅ Alto rendimiento: Ideal para aplicaciones que requieren velocidad (videojuegos, sistemas operativos).

✅ Control de hardware: Permite manipulación directa de memoria y recursos.

✅ Multiparadigma: Combina POO, programación genérica y procedural.

Desventajas de C++

❌ Curva de aprendizaje pronunciada: Complejo para principiantes.

❌ Gestión manual de memoria: Riesgo de memory leaks y errores de segmentación.

❌ Sintaxis compleja: Más verboso que lenguajes modernos (Python, Java).
