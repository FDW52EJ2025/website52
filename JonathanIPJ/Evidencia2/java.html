¿Que es Java?

Java es un lenguaje de programación orientado a objetos, robusto, portable y de alto nivel, diseñado para ser multiplataforma gracias a la JVM (Java Virtual Machine).

Caracteristicas de Java

✅Orientado a Objetos (POO): Usa clases, objetos, herencia, polimorfismo, encapsulamiento y abstracción.

✅Multiplataforma ("Write Once, Run Anywhere"): El código se compila en bytecode y se ejecuta en la JVM, compatible con Windows, Linux, macOS, etc.

✅Tipado Estático: Las variables deben declararse con su tipo antes de usarse.

✅Seguridad: Gestión de memoria estricta, sin punteros directos (evita accesos ilegales).

Ventajas de JavaScript

☑️Portabilidad: Funciona en cualquier sistema con JVM.

☑️Rendimiento: Más rápido que lenguajes interpretados (Python, JavaScript)

☑️Seguridad: Sin punteros explícitos, gestión de memoria controlada

Desventajas de JavaScript

❌Sintaxis más verbosa: Requiere más líneas de código que Python o JS.

❌Consumo de memoria: La JVM puede requerir más recursos

❌Tiempo de inicio lento: Aplicaciones pesadas pueden tardar en iniciar
