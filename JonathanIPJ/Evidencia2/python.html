Python

Python es un lenguaje de programación interpretado, multipropósito y de alto nivel, conocido por su sintaxis clara y legible. Es multiplataforma, orientado a objetos y soporta paradigmas de programación estructurada y funcional.

    Caracteristicas

  • Interpretado: No necesita compilación, se ejecuta línea por línea.

  • Multiplataforma: Funciona en Windows, Linux, macOS, etc.

  • Orientado a Objetos: Soporta programación orientada a objetos (POO).

  • Legible y Sencillo: Sintaxis clara y cercana al lenguaje humano.

Tipos de Datos

Tipo de Datos Ejemplos Descripcion
int 5 Números enteros (positivos o negativos).
float 3.14 Números decimales.
str "Hola" Cadenas de texto (entre comillas).
bool True / False Valores booleanos (verdadero/falso).
list [1, 2, 3] Listas (mutables y ordenadas).
tuple (1, 2, 3) Tuplas (inmutables y ordenadas).

    Ventajas

  • ☑️ Fácil de aprender: Ideal para principiantes.

  • ☑️ Versátil: Usado en web, IA, ciencia de datos, automatización, etc.

  • ☑️ Librerías potentes: Django (web), Pandas (datos), TensorFlow (IA).

    • Desventajas

  • ❌ Más lento que lenguajes compilados: Como C++ o Java (por ser interpretado).

  • ❌ Consumo de memoria: No es el mejor para aplicaciones de alto rendimiento.

  • ❌ Problemas con el tipado dinámico: Errores que solo aparecen en tiempo de ejecución.
