¿Que es JavaScript?

JavaScript es un lenguaje de programación interpretado, orientado a objetos y multiplataforma.

Caracteristicas de JavaScript

⭕Lenguaje del lado del cliente: Se ejecuta principalmente en navegadores web.

⭕Interpretado: No necesita compilación previa.

⭕Multiplataforma: Funciona en cualquier navegador y sistema operativo

    Datos Primitivos

  • Number: Números enteros y decimales (ej. 5, 3.14)
  • String: Cadenas de texto (ej. "Hola mundo")
  • Boolean: Valores lógicos true/false
  • null: Valor nulo intencional
  • undefined: Variable no definida
  • Symbol (ES6): Valor único e inmutable
  • BigInt (ES2020): Números enteros muy grandes

    • Tipos de Objetos

  • Object: Estructuras de datos clave-valor
  • Array: Listas ordenadas
  • Function: Bloques de código ejecutable
  • Date: Manipulación de fechas
  • RegExp: Expresiones regulares

Ventajas de JavaScript

☑️Ubicuidad: Funciona en todos los navegadores modernos sin necesidad de plugins

☑️Versatilidad: Se puede usar para frontend, backend (Node.js), móvil, escritorio, etc.

☑️Gran ecosistema: Miles de librerías y frameworks disponibles (React, Angular, Vue, etc.)

Desventajas de JavaScript

❌Rendimiento: Generalmente más lento que lenguajes compilados como C++ o Java

❌Seguridad: Vulnerable a inyección de código (XSS) en el lado del cliente

❌Coerción de tipos: El tipado débil puede causar comportamientos inesperados
