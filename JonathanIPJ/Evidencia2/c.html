¿Que es C?

C es un lenguaje de programación procedural, estructurado y de propósito general, desarrollado en 1972 por Dennis Ritchie en los laboratorios Bell. Es conocido como el "padre" de muchos lenguajes modernos.

Caracteristicas de C

✅ Lenguaje de bajo nivel: Permite acceso directo a memoria y hardware

✅ Eficiencia: Código compilado con alto rendimiento

✅ Portabilidad: Funciona en múltiples plataformas con mínimas modificaciones

✅ Estructurado: Soporta funciones y bloques de código

✅ Minimalista: Solo 32 palabras clave en el estándar original

✅ Manipulación directa de memoria: Uso de punteros

✅ Extensible: Permite creación de librerías y tipos de datos

Tipos de Datos

Tipo de Datos Descripción Ejemplo
int Entero con/sin signo (2-4 bytes) int count = 10;
float Número decimal precisión simple (4 bytes) float pi = 3.14;
double Número decimal doble precisión (8 bytes) double price = 9.99;
char Carácter ASCII (1 byte) char letter = 'A';
short Entero corto (2 bytes) short s = 100;
long Entero largo (4-8 bytes) long bigNum = 100000L;

Ventajas de C

✔ Rendimiento máximo: Casi tan rápido como el ensamblador

✔ Control absoluto: Acceso directo a hardware y memoria

✔ Lenguaje universal: Base para sistemas operativos y lenguajes modernos

✔ Eficiencia en recursos: Ideal para sistemas embebidos

✔ Gran comunidad: Documentación y soporte extensivos

Desventajas de C

✖ Gestión manual de memoria: Riesgo de leaks y corrupción

✖ No orientado a objetos: Falta de clases y herencia nativa

✖ Poca seguridad: Vulnerable a desbordamientos de buffer

✖ Curva de aprendizaje: Conceptos complejos como punteros

✖ Verbosidad: Requiere más código para tareas simples
