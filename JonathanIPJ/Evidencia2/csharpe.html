¿Que es C#?

C# (pronunciado "C Sharp") es un lenguaje de programación orientado a objetos, moderno y multipropósito, desarrollado por Microsoft como parte de la plataforma .NET.

Caracteristicas de C#

✅ Orientado a Objetos (POO): Usa clases, herencia, polimorfismo, encapsulamiento e interfaces.

✅ Lenguaje fuertemente tipado: Requiere declaración explícita de tipos.

✅ Multiplataforma: Gracias a .NET Core (ahora .NET 5+), funciona en Windows, Linux y macOS.

✅ Recolección de basura (Garbage Collector): Manejo automático de memoria.

✅ Sintaxis similar a Java y C++: Fácil de aprender para programadores con experiencia en estos lenguajes.

✅ Asincronía nativa: Soporte para async/await para operaciones no bloqueantes.

✅ LINQ (Language Integrated Query): Permite consultas integradas directamente en el código.

✅ Gran ecosistema: Amplias bibliotecas estándar y frameworks como ASP.NET, Xamarin, Unity, Blazor.

Ventajas de C#

✔ Productividad: Sintaxis limpia y herramientas como Visual Studio mejoran el desarrollo.

✔ Rendimiento: Más rápido que lenguajes interpretados (Python, JavaScript).

✔ Ecosistema .NET: Acceso a bibliotecas robustas para web, escritorio, móvil y juegos.

✔ Seguridad: Menos propenso a errores de memoria que C/C++ (gracias al GC).

✔ Multiplataforma: Con .NET 5+, ya no está limitado a Windows.

✔ Gran comunidad y soporte empresarial: Respaldado por Microsoft y ampliamente usado en la industria.

Desventajas de C#

✖ Dependencia de .NET: Aunque es multiplataforma, requiere el runtime de .NET.

✖ Menor rendimiento que C++: No es ideal para desarrollo de bajo nivel.

✖ Menos flexible que Python/JavaScript: No es el mejor para scripting rápido.

✖ Curva de aprendizaje para .NET: Dominar el ecosistema completo puede llevar tiempo.
