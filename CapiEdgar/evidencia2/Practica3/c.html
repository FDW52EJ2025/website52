💻 C

C es un lenguaje de programación estructurado, de bajo nivel y orientado a sistemas, que permite una gran eficiencia y control sobre el hardware. Es conocido por su sintaxis concisa y su capacidad para manipular directamente la memoria, lo que lo convierte en una excelente opción para sistemas operativos, aplicaciones embebidas y programación a nivel de hardware.

Características principales

  • Lenguaje de programación estructurado
  • Control directo sobre la memoria mediante punteros
  • Alta eficiencia y bajo consumo de recursos
  • Permite la manipulación directa de hardware
  • Extensa base de código heredada y ampliamente utilizado en sistemas operativos y aplicaciones embebidas

Tipos de datos

  • int: Números enteros
  • float: Números decimales de precisión simple
  • double: Números decimales de mayor precisión
  • char: Carácter o cadena de un solo carácter
  • void: Tipo de dato que no devuelve ningún valor

Ventajas

  • Excelente rendimiento debido a su bajo nivel
  • Control total sobre los recursos de hardware
  • Ampliamente utilizado en el desarrollo de sistemas operativos y aplicaciones embebidas
  • Permite una mayor flexibilidad y personalización

Desventajas

  • Gestión manual de memoria, lo que puede llevar a errores como fugas de memoria
  • Falta de características modernas como programación orientada a objetos
  • El código puede volverse complejo y difícil de mantener
  • Mayor propensión a errores de bajo nivel, como desbordamientos de búfer
