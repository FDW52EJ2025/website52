Python

Ventajas

  1. Sintaxis simple y fácil de leer
  2. Muchas librerías disponibles
  3. Gran comunidad de apoyo
  4. Funciona en cualquier sistema operativo

Desventajas

  1. Es más lento que otros lenguajes
  2. No es bueno para aplicaciones móviles
  3. Consume mucha memoria
  4. Dificultades para hacer programas ejecutables

Comentarios en Python

Los comentarios se hacen con el símbolo #:

# Esto es un comentario de una línea

Hola Mundo en Python

Así se escribe el programa más simple:

print("¡Hola, Mundo!")
