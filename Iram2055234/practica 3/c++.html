C++

Ventajas

  1. Alto rendimiento y velocidad
  2. Control preciso del hardware
  3. Ampliamente usado en desarrollo de juegos
  4. Multiplataforma

Desventajas

  1. Sintaxis compleja para principiantes
  2. Propenso a errores de memoria
  3. No tiene recolector de basura
  4. Curva de aprendizaje pronunciada

Comentarios en C++

Los comentarios se hacen con // o /* */:

// Esto es un comentario de una línea
/*
Esto es un comentario
de múltiples líneas
*/

Hola Mundo en C++
