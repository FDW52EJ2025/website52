JavaScript

Ventajas

  1. El único lenguaje que funciona en navegadores
  2. No necesita compilación
  3. Gran ecosistema (Node.js, React, etc.)
  4. Fácil de comenzar

Desventajas

  1. Comportamiento inconsistente entre navegadores
  2. Tipado débil puede causar errores
  3. Seguridad vulnerable si no se usa bien
  4. Difícil depuración en proyectos grandes
