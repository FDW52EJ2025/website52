Información General

C# es un lenguaje moderno, orientado a objetos, desarrollado por Microsoft como parte de su plataforma .NET.

Características Principales

  • Orientado a objetos
  • Compilado y con recolección de basura
  • Soporte para desarrollo de escritorio, web y móvil

Ventajas

  • Buena integración con herramientas de Microsoft
  • Lenguaje moderno y seguro
  • Multiplataforma con .NET Core

Desventajas

  • Dependencia del ecosistema Microsoft
  • Mayor consumo de recursos que otros lenguajes de bajo nivel
