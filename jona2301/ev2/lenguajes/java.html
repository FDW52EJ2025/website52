Información General

Java es un lenguaje de programación orientado a objetos diseñado para ser portable y seguro, ejecutándose en la JVM.

Características Principales

  • Orientado a objetos
  • Multiplataforma gracias a la máquina virtual
  • Fuertemente tipado

Ventajas

  • Gran portabilidad
  • Amplia comunidad y recursos
  • Buena integración empresarial

Desventajas

  • Verbosidad en el código
  • Requiere JVM para ejecutarse
