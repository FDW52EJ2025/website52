¿Qué es C?

C es un lenguaje de programación de propósito general, creado en 1972 por Dennis Ritchie en los laboratorios Bell. Es uno de los lenguajes más influyentes y ampliamente utilizados debido a su simplicidad, eficiencia y flexibilidad. C es conocido por ofrecer un control preciso sobre la memoria y el hardware, lo que lo hace ideal para desarrollar sistemas operativos, controladores de dispositivos, software embebido y aplicaciones que requieren alto rendimiento. Aunque su sintaxis es relativamente simple, su capacidad para manipular punteros y trabajar con estructuras de bajo nivel le otorgan gran potencia, aunque a costa de ser más propenso a errores si no se maneja adecuadamente

Características de C

  • Lenguaje de bajo nivel: Aunque se considera un lenguaje de alto nivel, C tiene características de lenguajes de bajo nivel que permiten un control más directo sobre el hardware.
  • Lenguaje estructurado: C permite una programación estructurada y modular, facilitando la organización y mantenimiento del código.
  • Portabilidad: C es un lenguaje muy portátil, lo que significa que el código escrito en C puede ser ejecutado en diferentes tipos de sistemas y plataformas con poca o ninguna modificación.
  • Control sobre el hardware: Ofrece una gran flexibilidad para manipular direcciones de memoria y trabajar con punteros, lo que permite optimizar el uso de los recursos del sistema.
  • Compilado: C es un lenguaje compilado, lo que significa que el código fuente es transformado a código máquina antes de ser ejecutado.
  • Sintaxis simple: La sintaxis del lenguaje es simple y directa, lo que facilita su aprendizaje y uso.

Tipos de Datos

Datos primitivos

int Tipo de dato entero, puede ser firmado (por defecto) o no firmado.

char Representa un carácter (generalmente almacenado en 1 byte.)

float Tipo de dato para representar números con decimales de precisión simple.

double Tipo de datos para números con decimales de precisión doble.

void Específica una función que no devuelve valor o un puntero sin tipo.

Datos derivador

Arrays Estructuras de datos que contienen múltiples elementos del mismo tipo.

Punteros Variables que almacenan direcciones de memoria.

structs Permiten agrupar diferentes tipos de datos bajo un mismo nombre.

unions Permiten almacenar diferentes tipos de datos en la misma posición de memoria.

enums Permiten crear tipos de datos con un conjunto específico de valores.

Ventajas y Desventajas de C

Ventajas

Eficiencia C es un lenguaje rápido, ya que permite un control preciso sobre la memoria y los recursos del sistema.

Portabilidad El código C es altamente portátil y puede ejecutarse en casi cualquier sistema operativo y plataforma con poca o ninguna modificación.

Control de bajo nivel C proporciona acceso directo a la memoria a través de punteros, lo que permite optimizar el rendimiento en aplicaciones que requieren alto rendimiento.

Amplio uso y documentación Es un lenguaje muy popular, con una extensa comunidad de usuarios, recursos educativos y bibliotecas disponibles

Desarrollo de sistemas operativos y software embebido Debido a su bajo nivel y control sobre el hardware, es ampliamente utilizado en la creación de sistemas operativos y software embebido.

Desventajas

Gestión manual de memoria C no tiene recolección automática de basura, lo que significa que el programador debe gestionar manualmente la memoria (asignarla y liberarla). Esto puede llevar a errores como fugas de memoria o acceso a memoria no válida.

SeguridadDebido a la falta de controles automáticos en la manipulación de punteros, es más fácil cometer errores que pueden resultar en vulnerabilidades de seguridad, como desbordamientos de búfer.

No es tan amigable con el desarrollador La sintaxis de C es más compleja en comparación con lenguajes más modernos, y las herramientas de desarrollo no son tan avanzadas como las de lenguajes más recientes.

Manejo de cadenas En C, las cadenas de texto se gestionan como arreglos de caracteres, lo que puede ser propenso a errores, como la falta de terminación de cadenas con el carácter \0.
