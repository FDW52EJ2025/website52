C

Informacion general

C es un lenguaje de programación de propósito general, de bajo nivel (comparado con otros lenguajes modernos), creado por Dennis Ritchie en los años 70 en los laboratorios Bell. Fue diseñado originalmente para el desarrollo del sistema operativo UNIX y desde entonces se ha convertido en uno de los lenguajes más influyentes de todos los tiempos. Muchos otros lenguajes, como C++, Java y Python, tienen su base en C. Es ideal para programación de sistemas, desarrollo de software embebido, y control directo del hardware.

Caracteristicas

Caracteristicas principales de C

  1. Lenguaje compilado y de alto rendimiento.
  2. Ofrece acceso directo a la memoria a través del uso de punteros.
  3. Sintaxis simple pero poderosa, cercana al lenguaje máquina.
  4. Soporta programación estructurada (uso de funciones y bloques).
  5. Es el lenguaje base para otros más modernos como C++, Java y C#.
  6. Muy utilizado en sistemas operativos, firmware y microcontroladores.

Tipos de datos

  1. Primitivos:t, double, char
  2. Modificadores: short, long, signed, unsigned
  3. Compuestos: arrays, struct, union
  4. Punteros: permiten manipular direcciones de memoria
  5. Enumeraciones: enum

Ventajas y Desventajas

Ventajas Desventajas
Muy rápido y eficiente, ideal para programación de bajo nivel. No tiene soporte nativo para programación orientada a objetos.
Es un lenguaje portátil, con la posibilidad de ser usado en casi cualquier plataforma. La depuración puede ser más complicada, especialmente en programas grandes.
Base fundamental para aprender otros lenguajes como C++ y Java. No cuenta con librerías modernas como otros lenguajes actuales.
Sintaxis clara y directa, sin abstracciones innecesarias. Menor seguridad en tiempo de ejecución si no se programan bien los controles de errores.
