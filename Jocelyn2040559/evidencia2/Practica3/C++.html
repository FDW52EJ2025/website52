C++

Informacion general

C++ es un lenguaje de programación de alto rendimiento, compilado y de propósito general, creado por Bjarne Stroustrup a principios de los años 80 como una extensión del lenguaje C. Está diseñado para ofrecer control total sobre el hardware y los recursos del sistema, manteniendo la eficiencia del lenguaje C pero incorporando características de la programación orientada a objetos, como clases, herencia y polimorfismo. C++ es ampliamente utilizado en el desarrollo de sistemas operativos, videojuegos, software embebido, controladores de hardware, aplicaciones de tiempo real y sistemas financieros.

Caracteristicas

  • - Lenguaje compilado, lo que lo hace rápido y eficiente.
  • - Soporta programación orientada a objetos (POO), pero también permite programación estructurada y genérica.
  • - Gran control sobre el uso de memoria, gracias al manejo de punteros.
  • - Compatible con C, lo que permite reutilizar librerías escritas en C.
  • - Usado en el desarrollo de software de alto rendimiento y bajo nivel.

Tipos de datos

  1. Primitivos: int, float, double, char, bool
  2. Modificadores: short, long, unsigned, signed
  3. Compuestos: arrays, struct, union
  4. Abstractos: class, enum, pointer, function

Ventajas y Desventajas

Ventajas Desventajas
Muy usado en desarrollo de videojuegos, motores gráficos y simuladores. El manejo manual de memoria puede causar errores como fugas o corrupción si no se hace correctamente.
Gran cantidad de herramientas, compiladores y librerías disponibles. Sintaxis más compleja y menos legible que lenguajes como Python.
Alto rendimiento: ideal para aplicaciones que requieren velocidad y eficiencia. No incluye un recolector de basura automático (a diferencia de lenguajes como Java o Python).
