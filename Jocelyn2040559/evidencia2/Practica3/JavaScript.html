Javascript

Informacion general

JavaScript es un lenguaje de programación de alto nivel, interpretado y basado en objetos, creado inicialmente por Brendan Eich en 1995 como parte de la web para añadir interactividad a los navegadores. JavaScript es esencial en el desarrollo web, permitiendo la creación de páginas dinámicas, interactivas y la manipulación del contenido HTML y CSS. Es el lenguaje principal para la programación del lado del cliente (aunque también se usa en el lado del servidor gracias a Node.js). JavaScript es el lenguaje estándar en la web, funcionando en todos los navegadores modernos y soportado por numerosos frameworks y bibliotecas como React, Angular y Vue.js.

Caracteristicas

  1. Interpretado: el código se ejecuta línea por línea en el navegador o en un servidor (en el caso de Node.js).
  2. Lenguaje orientado a objetos y basado en prototipos: no usa clases tradicionales (aunque las clases fueron introducidas en ES6).
  3. Soporta event-driven programming (programación basada en eventos).
  4. Compatible con todos los navegadores web y tiene acceso directo al DOM (Document Object Model) para manipular el contenido de las páginas web.
  5. Asíncrono: soporta callbacks, promesas y async/await para operaciones no bloqueantes.
  6. Ampliamente utilizado para desarrollo full-stack con Node.js (servidor) y React/Angular/Vue (front-end).

Tipos de datos

  1. Primitivos: string, number, boolean, null, undefined, symbol, bigint
  2. Objetos: object, array, function
  3. Especiales: NaN (Not-a-Number), Infinity

Ventajas y Desventajas

Ventajas Desventajas
Lenguaje fundamental para el desarrollo web, compatible con todos los navegadores. Aunque se puede escribir código asíncrono, los callbacks anidados pueden resultar difíciles de manejar
Fácil de aprender para principiantes, con una amplia comunidad de soporte. Problemas con el manejo de errores: el sistema de tipos dinámicos puede llevar a errores difíciles de depurar.
Soporta programación funcional y orientada a objetos (con funciones de orden superior, callbacks, etc.). No es tan rápido como lenguajes compilados como C o C++.
Gran ecosistema de herramientas y bibliotecas como React, Angular, Node.js, etc. No es adecuado para aplicaciones de alto rendimiento como juegos complejos o aplicaciones de tiempo real exigentes.
