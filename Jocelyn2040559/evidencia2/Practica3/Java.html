Java

Informacion general

Java es un lenguaje de programación orientado a objetos que se utiliza para crear aplicaciones y software. Es multiplataforma, lo que significa que se puede ejecutar en distintos sistemas operativos.

Java es un lenguaje de programación de alto nivel, orientado a objetos y de propósito general, creado por James Gosling en 1995 para Sun Microsystems (ahora propiedad de Oracle). Su lema es "write once, run anywhere" (escribe una vez, ejecútalo en cualquier lugar), gracias a su funcionamiento sobre la Java Virtual Machine (JVM), que permite que los programas Java se ejecuten en múltiples plataformas sin necesidad de modificarlos. Java es muy utilizado en aplicaciones empresariales, aplicaciones móviles (especialmente Android), desarrollo web del lado del servidor y sistemas embebidos.

Caracteristicas

    Características principales de Java
  1. - Lenguaje orientado a objetos: todo está basado en clases y objetos.
  2. - Portabilidad multiplataforma a través de la JVM.
  3. - Sintaxis influenciada por C/C++, pero más simple y segura.
  4. -Incluye un recolector de basura automático para la gestión de memoria.
  5. - Posee una gran cantidad de librerías y frameworks (Spring, Hibernate, JavaFX, etc.).

Tipos de datos

  1. Primitivos: int, float, double, char, boolean, byte, short, long
  2. Referenciados: String, Arrays, Classes, Interfaces, Enums, etc.
  3. Colecciones: List, Set, Map, etc. (a través de la biblioteca Collections)

Ventajas y Desventajas

Ventajas Desventajas
Portabilidad: un mismo código puede ejecutarse en cualquier dispositivo con JVM. Consumo de memoria y recursos más alto comparado con lenguajes como C++.
Lenguaje seguro con control de errores en tiempo de compilación y ejecución. Velocidad de ejecución más lenta que lenguajes compilados directamente a código máquina.
Gran ecosistema de librerías, herramientas y frameworks disponibles. La sintaxis puede parecer verbosa, especialmente en comparación con Python.
Muy usado en el desarrollo de aplicaciones Android. La JVM puede tener problemas de compatibilidad entre versiones.
