Características Principales

  • Orientado a objetos (parte de .NET).
  • Compilado a bytecode (ejecutado en CLR).
  • Usado en apps Windows, Unity y desarrollo empresarial.
  • Recolección de basura automática.

Tipos de Datos

  • Primitivos: int, bool, decimal.
  • Objetos: string, List, Dictionary.

Ventajas

  • Sintaxis moderna y similar a Java.
  • Ideal para desarrollo de videojuegos (Unity).

Desventajas

  • Menos multiplataforma que Java (aunque .NET Core lo mejora).
  • Más ligado a ecosistemas Microsoft.
