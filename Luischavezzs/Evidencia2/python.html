Python

Características principales

  • Lenguaje interpretado (se ejecuta línea por línea)
  • Sintaxis sencilla y legible
  • Tipado dinámico
  • Multiparadigma (estructurado, orientado a objetos, funcional)
  • Gran comunidad y muchas bibliotecas
  • Portabilidad entre sistemas operativos
  • Lenguaje de alto nivel
  • Gestión automática de memoria

Tipos de datos

  • Numéricos: int, float, complex
  • Texto: str
  • Booleanos: bool (True o False)
  • Secuencias: list, tuple, range
  • Conjuntos: set, frozenset
  • Mapas: dict (diccionario clave-valor)

Ventajas

  • Fácil de aprender y usar
  • Gran cantidad de bibliotecas (NumPy, Pandas, etc.)
  • Ideal para prototipado rápido
  • Muy versátil (web, IA, ciencia de datos, etc.)
  • Buena integración con otros lenguajes

Desventajas

  • Más lento que lenguajes compilados
  • Alto consumo de memoria
  • No recomendado para apps móviles o en tiempo real
  • Errores detectados en tiempo de ejecución
