Lenguaje C

Características

  • C es conocido por su simplicidad y eficiencia. Permite a los programadores escribir código de bajo nivel.

  • Una de las grandes ventajas de C es su portabilidad. El código escrito en C puede ser compilado y ejecutado en diferentes plataformas sin cambios importantes.

  • C ofrece un alto nivel de control sobre el hardware y los recursos del sistema.

  • El lenguaje C cuenta con una rica biblioteca estándar que proporciona una amplia gama de funciones para realizar tareas comunes,.

Tipos de datos

  1. Numéricos enteros: int, short, long, long long

  2. Numéricos en punto flotante: float, double, long double

  3. Caracteres: char

  4. Booleanos: _Bool (desde C99)

  5. Apuntadores: int*, char*, void*

  6. Arreglos: int[], char[], float[]

  7. Estructuras: struct

  8. Uniones: union

  9. Enumeraciones: enum

Ventajas y desventajas

  • Ventajas:

  • Es un lenguaje rápido y eficiente en el uso de recursos.

  • Proporciona un alto nivel de control sobre la memoria.

  • Es un lenguaje portable y se puede compilar en múltiples plataformas.

  • Es la base de muchos otros lenguajes de programación modernos.

  • Cuenta con una amplia comunidad y mucha documentación disponible.

  • Desventajas:

  • No tiene una gestión automática de memoria (no cuenta con Garbage Collector).

  • El manejo de punteros puede ser complicado y propenso a errores.

  • No tiene soporte nativo para la programación orientada a objetos.

  • No cuenta con una biblioteca estándar extensa como otros lenguajes modernos.
