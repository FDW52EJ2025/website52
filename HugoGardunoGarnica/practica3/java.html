Java

Características

  • Java es orientado a objetos, lo que significa que está organizado en torno a objetos en lugar de "eventos" y datos en lugar de lógica.

  • Java es independiente de la plataforma, lo que significa que puede ejecutarse en cualquier sistema operativo que admita una máquina virtual Java (JVM).

  • Java es seguro porque utiliza la Arquitectura de Seguridad Java (JSA) para proteger contra el código malicioso.

  • Java es robusto porque incorpora la gestión de excepciones y la recolección de basura.

Tipos de datos

  1. Numéricos: byte, short, int, long, float, double

  2. Caracteres: char

  3. Cadenas: String

  4. Booleanos: boolean

  5. Arreglos: int[], double[], String[]

  6. Clases y Objetos: Cualquier clase creada por el usuario

  7. Colecciones: List, Set, Map (a través de interfaces y clases como ArrayList, HashSet, HashMap)

Ventajas y desventajas

  • Ventajas:

  • Es multiplataforma gracias a la Máquina Virtual de Java (JVM).

  • Es un lenguaje orientado a objetos, lo que facilita la organización del código.

  • Cuenta con recolección de basura automática (Garbage Collector).

  • Tiene una gran comunidad y muchas bibliotecas disponibles.

  • Es un lenguaje seguro gracias a su gestión de memoria y manejo de excepciones.

  • Desventajas:

  • Es más lento que los lenguajes compilados como C++ debido a la JVM.

  • El consumo de memoria es mayor en comparación con otros lenguajes.

  • La sintaxis puede ser más verbosa en comparación con lenguajes como Python.

  • No es la mejor opción para aplicaciones de tiempo real o sistemas embebidos.
