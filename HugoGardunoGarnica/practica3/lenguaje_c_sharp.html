Lenguaje C#

Características

  • C# es un lenguaje de programación orientado a objetos ya que sigue los tres principios de este modelo, la encapsulación, la herencia y el polimorfismo.

  • El lenguaje incluye una serie de reglas o normas de sintaxis para controlar el acceso a los tipos de datos.

  • No es necesario incluir instrucciones de destrucción de objetos en el código ya que el recolector de basura del CLR lo hará por nosotros.

  • En C# se unifican todos los tipos de datos ya que estos derivan de la clase común.

Tipos de datos

  1. Numéricos enteros: byte, sbyte, short, ushort, int, uint, long, ulong

  2. Numéricos en punto flotante: float, double, decimal

  3. Caracteres: char

  4. Cadenas: string

  5. Booleanos: bool

  6. Arreglos: int[], char[], double[]

  7. Enumeraciones: enum

  8. Estructuras: struct

  9. Clases y Objetos: Cualquier clase definida por el usuario

  10. Colecciones: List, Dictionary, HashSet, Queue, Stack

  11. Apuntadores (solo en código inseguro): int*, char*

  12. Tipos dinámicos: dynamic

Ventajas y desventajas

  • Ventajas:

  • Es un lenguaje orientado a objetos, lo que facilita la organización del código.

  • Cuenta con una recolección automática de basura (Garbage Collector) para la gestión de memoria.

  • Es un lenguaje multiplataforma gracias a .NET Core.

  • Proporciona una sintaxis clara y fácil de leer.

  • Tiene una gran comunidad y documentación extensa.

  • Cuenta con un alto nivel de seguridad gracias a su tipado fuerte.

  • Compatible con desarrollo de aplicaciones de escritorio, móviles, web y videojuegos (Unity).

  • Desventajas:

  • Puede ser más lento que C o C++ debido a la ejecución en la Máquina Virtual de .NET (CLR).

  • Requiere el entorno de ejecución .NET para funcionar.

  • No tiene acceso directo a la memoria (excepto en código inseguro con punteros).

  • El desarrollo en C# generalmente está más ligado a tecnologías de Microsoft.
