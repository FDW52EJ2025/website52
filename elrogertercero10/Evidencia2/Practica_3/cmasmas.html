A grandes rasgos, ¿qué es C++?

C++ es un lenguaje de programación de propósito general, orientado a objetos y de bajo nivel, que se basa en el lenguaje C, pero añade características como clases, herencia, polimorfismo y control directo sobre la memoria. Es ampliamente utilizado para el desarrollo de software de alto rendimiento, aplicaciones de sistemas, videojuegos, y software embebido.

¿Qué hace diferente a C++ del resto de lenguajes de programación?

Programación orientada a objetos (OOP):
Soporta conceptos como clases, objetos, herencia, polimorfismo y encapsulación.
Control de memoria:
Permite un control detallado de la memoria mediante punteros, lo que lo hace eficiente pero también susceptible a errores.
Lenguaje compilado:
El código C++ se compila a código máquina, lo que generalmente resulta en programas más rápidos y eficientes en comparación con los lenguajes interpretados.
Funciones estándar y plantillas:
C++ ofrece una biblioteca estándar (STL) y permite la programación genérica a través de plantillas.

Tipos de datos

  • Número
      • int: Enteros.
      float: Números con punto flotante de precisión simple.
      double: Números con punto flotante de doble precisión.
      long: Enteros más grandes.
  • char (Caracteres): Un sólo caracter.
  • struct (Arreglos): Estructuras para agrupar datos de diferentes tipos.
  • pointer (Punteros): Variables que almacenan la dirección de memoria de otras variables.

Ventajas y desventajas de C++

Ventajas

Portabilidad:
C++ es compatible con muchas plataformas, lo que permite ejecutar el mismo código en diferentes sistemas operativos y hardware.
Soporte para programación orientada a objetos y genérica:
La capacidad de C++ para utilizar clases, herencia, polimorfismo, y plantillas hace que sea extremadamente versátil y eficiente.

Desventajas

Complejidad en la gestión de memoria:
El control manual de la memoria (a través de punteros) puede llevar a errores como fugas de memoria, lo que dificulta la depuración.
Curva de aprendizaje empinada:
C++ es más complejo que lenguajes como Python debido a sus características avanzadas, como punteros, manejo explícito de memoria, y la sintaxis de programación orientada a objetos.
