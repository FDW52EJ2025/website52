Lenguaje C

Es un lenguaje de programación de propósito general, estructurado y de bajo nivel. Fue desarrollado por Dennis Ritchie en los laboratorios Bell en la década de 1970. Es ampliamente utilizado para el desarrollo de sistemas operativos, software embebido y aplicaciones de alto rendimiento.

Características principales de C

• Lenguaje estructurado: Permite dividir el programa en funciones y bloques lógicos.

• Bajo nivel: Permite un control cercano al hardware.

• Portabilidad: El código C puede ser compilado en múltiples plataformas con mínimas modificaciones.

• Eficiencia: Tiene una gran velocidad de ejecución y bajo consumo de memoria.

• Lenguaje compilado: El código fuente debe ser compilado antes de su ejecución.

Tipos de datos en C

➨ Enteros (int): Representan números enteros, tanto positivos como negativos, sin decimales.

➨ Flotantes (float): Representan números con decimales.

➨ Caracteres (char): Representan caracteres individuales, como letras, dígitos y símbolos.

➨ Booleanos (bool): C no tiene un tipo booleano incorporado como tal en su estándar original, a partir de C99, se incluyó el tipo _Bool.

Ventajas

✅ Alto rendimiento: Ideal para programación de sistemas.

✅ Control total: Acceso a memoria y hardware.

✅ Portabilidad: Muy utilizado en distintos sistemas operativos.

Desventajas

❌ Mayor complejidad: Sintaxis más técnica y menos intuitiva.

❌ Gestión manual de memoria: Mayor riesgo de errores como fugas de memoria.

❌ No soporta programación orientada a objetos de forma nativa.
