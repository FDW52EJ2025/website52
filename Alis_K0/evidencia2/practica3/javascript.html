Lenguaje JavaScript

Es un lenguaje de programación interpretado, ligero y orientado a objetos, ampliamente utilizado para el desarrollo web. Fue creado por Brendan Eich en 1995 y permite agregar interactividad y dinamismo a las páginas web. Es un componente clave del desarrollo frontend junto con HTML y CSS.

Características principales de JavaScript

• Lenguaje interpretado: Se ejecuta directamente en el navegador sin necesidad de compilación.

• Dinámico: Permite modificar el contenido de la página en tiempo real.

• Multiparadigma: Soporta programación orientada a objetos, funcional y basada en eventos.

• Integración con HTML/CSS: Trabaja en conjunto con otras tecnologías web.

• Compatible con todos los navegadores modernos: Es esencial para el desarrollo web.

Tipos de datos en JavaScript

➨ Primitivos: son aquellos que no tienen propiedades ni métodos, y son inmutables. En JavaScript, existen seis tipos de datos primitivos: Number, String, Boolean, Null, Undefined y BigInt.

➨ Compuestos: os tipos de datos compuestos, también conocidos como tipos de datos estructurados o de referencia, son aquellos que pueden contener múltiples valores y tienen propiedades y métodos. En JavaScript, los principales tipos de datos compuestos son Object, Array, Function y Date.
