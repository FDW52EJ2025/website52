Lenguaje Java

Es un lenguaje de programación orientado a objetos, desarrollado por Sun Microsystems y lanzado en 1995. Es ampliamente utilizado para el desarrollo de aplicaciones empresariales, móviles (especialmente en Android), web y sistemas integrados. Su lema "escribe una vez, ejecuta en cualquier lugar" refleja su portabilidad.

Características principales de Java

• Orientado a objetos: Facilita la reutilización y organización del código a través de clases y objetos.

• Portabilidad: El código Java se compila en bytecode, que puede ejecutarse en cualquier sistema con la máquina virtual de Java (JVM).

• Seguridad: Incluye mecanismos para evitar errores y ataques de seguridad.

• Multihilo: Permite la ejecución de múltiples procesos al mismo tiempo.

• Gestión automática de memoria: Usa recolector de basura para liberar memoria no utilizada.

Tipos de datos en Java

➨ Primitivos: Representan valores simples y se almacenan directamente en la memoria: enteros, numeros de punto flotante, caracteres y booleanos

➨ No primitivos: Son aquellos que son objetos y tienen métodos. En Java, hay varios tipos de datos no primitivos diferentes: string, arrays y clases.

➨ Referencias: son aquellos que se utilizan para referirse a objetos en lugar de almacenar los objetos en sí mismos. Estos tipos de datos se utilizan para manipular objetos y pasarlos entre diferentes partes de un programa: object, class e interface.

Ventajas

✅ Alta portabilidad gracias a la JVM.

✅ Lenguaje robusto y seguro.

✅ Amplio soporte para desarrollo móvil (Android).

✅ Gran comunidad y amplia documentación.

Desventajas

❌ Consumo de memoria y rendimiento más bajo que lenguajes como C++.

❌ Verbosidad: el código puede ser más extenso en comparación con otros lenguajes.

❌ Arranque más lento debido a la JVM.
