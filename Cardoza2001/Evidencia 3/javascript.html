JavaScript - Lenguaje de Programación

Definición

JavaScript es un lenguaje de programación de alto nivel utilizado principalmente en la creación de sitios web interactivos. Fue desarrollado por Brendan Eich en 1995 y es uno de los tres lenguajes fundamentales de la web junto con HTML y CSS.

Beneficios

  • Permite interactividad en sitios web
  • Compatible con todos los navegadores
  • Amplia comunidad y recursos
  • Uso en front-end y back-end (Node.js)

Características

  • Lenguaje de alto nivel
  • Asíncrono y basado en eventos
  • Lenguaje interpretado en el navegador
  • Soporta programación funcional y orientada a objetos
