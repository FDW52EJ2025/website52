C - Lenguaje de Programación

Definición

El lenguaje C es uno de los lenguajes de programación más antiguos y ampliamente utilizados. Fue desarrollado en 1972 por Dennis Ritchie para el sistema operativo Unix. Es un lenguaje de bajo nivel que permite un control completo del hardware.

Beneficios

  • Control total sobre el hardware
  • Portabilidad entre plataformas
  • Ampliamente utilizado en sistemas embebidos
  • Velocidad de ejecución muy alta

Características

  • Lenguaje de bajo nivel
  • Sintaxis sencilla
  • Facilita la manipulación de memoria
  • Lenguaje estructurado
